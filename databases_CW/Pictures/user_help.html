Руководство пользователя

Автоматизированная информационная система для мелкооптового книжного магазина

О программе

Данная программа - это автоматизированная информационная система для мелкооптового книжного магазина, разработанная для управления базой данных. Программа состоит из нескольких функциональных разделов, каждый из которых предназначен для решения конкретных задач управления магазином.

Основное назначение: управление базой данных книжного магазина, включая работу с товарами, заказами, поставками и персоналом.

Разделы программы

Программа состоит из следующих разделов:

Раздел Доступность Описание
Справочники Все пользователи Базовая справочная информация, необходимая для работы системы
Документы Все пользователи Работа с документацией и отчетами
Справка Все пользователи Руководство пользователя и информация о программе
Разное Все пользователи Дополнительные инструменты и настройки
Заказы По ролям Управление заказами клиентов
Книги По ролям Управление каталогом книг и ассортиментом
Поставки По ролям Управление поставками от поставщиков
Отдел кадров По ролям Управление персоналом и кадровые операции

Примечание: Разделы "Справочники", "Документы", "Справка" и "Разное" доступны всем пользователям. Остальные разделы доступны только для определенных ролей пользователей.

Функциональные возможности

Программа поддерживает следующие основные функции:

  • Просмотр всех таблиц - возможность просматривать все таблицы базы данных в удобном интерфейсе
  • CRUD-операции - создание, чтение, обновление и удаление записей (доступны в зависимости от ролей пользователей)
  • Работа с SQL-запросами - просмотр типовых SQL-запросов и возможность написания собственных. Результаты запросов можно скачать в формате HTML-файла

Дополнительные возможности

В программе также реализованы следующие разделы:

Раздел Функциональность
Справка Содержит руководство пользователя и информацию о программе
Личный кабинет Настройка программы и смена пароля пользователя

Для получения дополнительной информации или помощи в использовании программы обратитесь к администратору системы.

Помните: некоторые функции доступны только пользователям с соответствующими ролями.
